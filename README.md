# English Learning Chatbot

A chatbot built with Flask that helps learners find English practice activities.

Features:
- Grammar, vocabulary and exam practice recommendations
- Topic detection (conditionals, reported speech, etc.)
- Clickable activity links
- Simple conversational responses

Tech stack:
- Python
- Flask
- HTML / JavaScript

Future improvements:
- Firebase integration
- smarter NLP detection
- lesson recommendations
